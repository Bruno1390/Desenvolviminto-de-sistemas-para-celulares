Questões

1. Defina dispositivo móveis.

R.Dispositivos móveis são tecnologias digitais que permitem a mobilidade e o acesso à internet. Pode-se citar como exemplos os smartphones e tablets.

2. Qual o papel da internet no atual avanço tecnológico ?

R.Ela possibilitou que relações fossem mantidas mesmo que à distância, contribuiu para o fácil acesso e divulgação de informações importantes, ajudou para que negócios sobrevivessem e outros até mesmo crescessem ainda mais, como foi o caso do e-commerce.

3. Cite os principais tipos de dispositivos móveis

R.Podemos citar alguns exemplos de dispositivos móveis: smartphone, ultrabook, notebook, netbook, GPS, tablet.

4. Para que serve o nível de aplicação da arquitetura android?

R.A arquitetura do Android segue fortemente conceitos de sandboxing e abstração, permitindo alto nível de segurança e estabilidade, enquanto simplifica o desenvolvimento dos aplicativos. Apesar de ter como base o Kernel do Linux, o Android funciona de uma maneira muito diferente, devido ao fator móvel que possui.

5. O SO do andorid é baseado no Linux. Qual seu papel na arquitetura do andorid

R.O linux é a base da arquitetura do Android, fornecendo o kernel do sistema operacional que controla o acesso ao hardware e gerencia os recursos do sistema

6. Qual a principal característica do android?

R.Android é um sistema operacional (SO) baseado no núcleo Linux, projetado principalmente para dispositivos eletrônicos móveis (como smartphones e tablets) com tela sensível ao toque ou interface de usuário baseada na manipulação direta; desenvolvido por um consórcio de desenvolvedores conhecido como Open Handset ...

7. Qual a diferença de um aplicativo nativo para um Desenvolvimento por terceiro ?

R.Os aplicativos nativos são capazes de proporcionar aos usuários uma experiência mais intensa do que os híbridos. Porque eles são desenvolvidos especialmente para funcionar em determinado sistema operacional e oferecer melhores recursos de hardware, facilitando a utilização de funções e recursos próprios do app.

8. Como o andorid se diferencia de outros sistemas de dispositivos móveis ?

R.Uma das principais diferenças entre os sistemas é a disponibilidade. Desenvolvido pelo Google, o Android não é exclusivo e equipa vários modelos de diferentes marcas, entre elas gigantes como Samsung, Huawei, LG e Asus, por exemplo, além de sua linha de celulares, a Pixel.

9. Como funciona o nível de Frameword ds arquitetura do android?

R. esta camada é desenvolvida quase toda em Java, e faz a interface com as aplicações Android. Ela provê um conjunto de bibliotecas para acessar os diversos recursos do dispositivo como interface gráfica, telefonia, localizador (GPS), banco de dados persistente, armazenamento no cartão SD, etc.

10. E o nível de bibliotecas ?

R.Existem formas diferentes de catalogação, mas a mais utilizada é a Classificação Decimal de Dewey – CDD, para os íntimos – comum na maioria das bibliotecas em 135 países. Ela divide o conhecimento humano em dez grandes classes e atribui um número para cada uma.
